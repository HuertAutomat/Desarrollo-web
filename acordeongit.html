Acordeon de comandos GIT

COMANDS BÁSICOS

cd
Comando para cambiar la carpeta
ls
Comando que muestra un istado de los elementos en la carpeta acual
mkdir
Comando para crear una nueva carpeta
pwd
Comando que indica en que carpeta te encuentras
cp
Comando para crear archivos o carpetas
mv
Comando para mover archivos o carpetas
rm
Comando para eliminar archivos. Para eliminar carpetas se usa rm -r>/
